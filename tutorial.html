Tutorial for using ZEOGS

1. Getting the data

In this tutorial, we will use a dataset containing a list of genes, whose expression was shown to be dependent on Hedgehog signaling, an important signaling pathway essential for successful development of several organ systems. The data itself comes from the following paper:

Xu J, Srinivas BP, Tay SY, Mak A, Yu X, Lee SG, Yang H, Govindarajan KR, Leong B, Bourque G, Mathavan S, Roy S.
Genomewide expression profiling in the zebrafish embryo identifies target genes regulated by Hedgehog signaling during vertebrate development. Genetics. 2006 Oct;174(2):735-52.

The genes in the list had their expression down-regulated by cyclopamine, a Hedgehog signaling inhibitor, and up-regulated by over-expression of the dominant-active Protein Kinase A, a potent activator of the Hedgehog signaling. These genes are thus likely target genes of transcription factors downstream in the Hedgehog signaling pathway. Please download a file with these genes here.

2. Analyzing the data

To analyze the gene list you have downloaded above you need to do the following steps

  1. 1. Open a fresh ZEOGS form page.

  2. 2. Copy the genes in the file you have downloaded.

  3. 3. Leave the data source option unchanged, for the gene identifier format select ZFIN symbol.

  4. 4. Paste the previously copied gene symbols into the text input area.

  5. 5. For the stage option, select Pharyngula:Prim-5 - 24-30 hours because this is the stage when the experiments in the paper were performed.

  6. 6. For the sorting option, please select P-value: ascending order.

  7. 7. Now you are ready to press the Submit button and run the analysis.

For the discussion of all the options in this program please refer to the Help section
