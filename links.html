Links to sites providing gene expression pattern information:

  • ZFIN

    • The central repository of information concerning zebrafish as a model system.

  • Ensembl

    • Genome browser platform for multiple species with the access to BioMart for database queries and data set processing.

  • 4DXpress

    • A database containing expression pattern information for genes in multiple species allowing comparisons of gene expression pattern during evolution.

  • MEPD

    • A database of gene expression patterns in Medaka fish.

  • MGI

    • The central repository of information concerning mouse as a model system.

  • Gene Paint

    • A database of expression pattern information from in situ hybridisation experiments for the mouse.

  • Emouseatlas

    • A website containing several tools such as Mouse Anatomy Atlas and Mouse Gene Expression Spatial Database.

  • BDGP

    • Berkeley Drosophila Genome Project website with a lot of genomic and expression information for the Drosophila community.
